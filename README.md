This is a movie poster web application. This is Next.js project with Feature-Sliced Design architecture. The project uses Tailwind, Redux Toolkit.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
